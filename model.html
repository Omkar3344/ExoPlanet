Module A: Habitable Zone Boundaries

Calculates the inner and outer boundaries of the habitable zone based on stellar flux and applies eccentricity corrections for elliptical orbits.

Compute Seff (effective stellar flux) for each planet
Calculate distance for Inner HZ (IHZ) and Outer HZ (OHZ)
Apply eccentricity correction using e = √(1 - b²/a²) 
def calculate_hz_boundaries(star_temp, star_luminosity):
    # Conservative HZ boundaries (Kopparapu et al. 2013)
    S_eff_inner = 1.776  # Recent Venus
    S_eff_outer = 0.32   # Early Mars
    
    # Calculate distances
    d_inner = sqrt(star_luminosity / S_eff_inner)
    d_outer = sqrt(star_luminosity / S_eff_outer)
    
    return d_inner, d_outer

Module C: Atmospheric Mass Loss

Implements energy-limited atmospheric escape models to determine if planets can retain atmospheres conducive to life.

Energy-limited mass-loss formula implementation
Compare escape timescale (tloss) to system age
Account for XUV flux from host star 
def calculate_mass_loss(planet_mass, planet_radius, 
                                        stellar_xuv, distance):
    # Energy-limited escape rate
    efficiency = 0.15  # Typical value
    
    mass_loss_rate = (efficiency * stellar_xuv * pi * 
                      planet_radius**3) / (G * planet_mass * distance**2)
    
    # Escape timescale
    t_loss = planet_mass / mass_loss_rate
    
    return t_loss

Module D: Tidal Heating

Calculates tidal heat flux from orbital parameters to identify planets in the "tidal habitable zone" where subsurface oceans might exist.

Calculate tidal heat flux from orbital eccentricity
Consider tidal dissipation factor (Q)
Flag planets in IHZ ∩ tidal-HZ overlap

Tidal Heating Formula

Ptidal = (21/2) × (k₂/Q) × (n⁵ R⁵ e²) / G

Where k₂ is Love number, Q is tidal dissipation factor, n is mean motion, R is radius, e is eccentricity
